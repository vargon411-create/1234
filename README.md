# Личная кинотека

## Автор
Ярцев Илья

## Описание
Приложение «Movie Library» позволяет пользователю добавлять, фильтровать и сохранять фильмы в формате JSON. Пользователь может хранить информацию о названии, жанре, годе выпуска и рейтинге фильма.

## Примеры использования
1. Введите название, жанр, год выпуска и рейтинг фильма, затем нажмите кнопку «Добавить фильм».
2. Для фильтрации фильмов по жанру или году ввода введите нужные значения и нажмите кнопку «Фильтровать».
3. Все фильмы сохраняются в файл `movies.json` для последующего использования.
