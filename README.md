# Random Quote Generator

## Автор
Ярцев Илья

## Описание
Приложение «Генератор случайных цитат» позволяет генерировать случайные цитаты из предопределённого списка, а также сохраняет и отображает историю сгенерированных цитат.

## Примеры использования
1. Нажмите кнопку «Сгенерировать цитату» для получения новой цитаты.
2. Используйте поля для фильтрации цитат по автору и теме.
3. История сгенерированных цитат сохраняется на диске.
